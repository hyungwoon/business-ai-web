# Talent Management Skill

Frameworks for recruiting, performance management, compensation, and organizational development in Korean and global business contexts.

## Recruiting Framework

### Job Description Template

```
직책명 (Job Title):
팀/부서 (Team/Department):
고용 형태 (Employment Type): 정규직 / 계약직 / 인턴
근무지 (Location):
보고 라인 (Reports to):

[회사 소개 — 2-3문장, 미션 + 현재 단계]

역할 개요 (Role Summary):
[이 역할이 존재하는 이유와 핵심 임팩트 — 3-4문장]

주요 업무 (Key Responsibilities):
- [동사로 시작하는 구체적 업무 — 5-7개]

필수 자격 (Required Qualifications):
- [측정 가능한 경험/스킬 — 3-5개]

우대 사항 (Nice to Have):
- [선호 요소 — 3개 이하]

보상 패키지 (Compensation):
- 연봉: [범위 또는 "경력에 따라 협의"]
- 스톡옵션: [있을 경우 규모]
- 복리후생: [핵심 항목 3-5개]
```

### Interview Process Design

**4단계 표준 프로세스:**

| 단계 | 형식 | 목적 | 소요 시간 |
|---|---|---|---|
| 1. 서류 검토 | 비동기 | 기본 자격 필터링 | - |
| 2. 인재 스크리닝 | 30분 화상 | 동기, 커뮤니케이션 평가 | HR |
| 3. 직무 인터뷰 | 1-2시간 | 기술/역량 심층 평가 | 팀장 + 팀원 |
| 4. 임원 면접 | 30-45분 | 문화 적합도, 최종 확인 | C레벨 |

**역량 기반 인터뷰 질문 구조 (STAR):**
- **Situation:** 상황을 설명해주세요
- **Task:** 당신의 역할/목표는 무엇이었나요?
- **Action:** 구체적으로 어떤 행동을 취했나요?
- **Result:** 결과는 어땠나요? 수치로 설명해주세요

**평가 루브릭 (1-5점):**

| 역량 | 1 (미달) | 3 (충족) | 5 (탁월) |
|---|---|---|---|
| 문제 해결 | 지시 없이는 해결 못함 | 독립적으로 해결 | 새로운 접근법 제시 |
| 커뮤니케이션 | 의견 전달 불명확 | 명확하게 전달 | 설득력 있게 영향 |
| 오너십 | 외부 귀인 경향 | 본인 영역 책임 | 조직 전체에 기여 |

## 성과 관리 (Performance Management)

### OKR 작성 가이드

**Objective 작성 원칙:**
- 영감을 주는 방향성 제시 (측정 불필요)
- 분기 단위, 팀이 자율적으로 달성 가능
- 예: "고객이 우리 제품 없이는 살 수 없게 만든다"

**Key Result 작성 원칙:**
- 구체적, 측정 가능, 기간 명시
- Objective당 3-5개
- 0-1.0 스케일로 진행률 측정
- 예: "NPS를 42에서 58로 향상" / "월 MAU 10만 달성"

**흔한 실수:**
- KR에 활동(Activity)을 적는 것 → 결과(Outcome)로 수정
- KR이 너무 쉬움 (100% 달성 예상) → 70% 달성 목표로 설정
- 개인 OKR이 팀 OKR과 연결 안 됨

### 성과 리뷰 사이클

**반기 리뷰 구조:**

```
자기 평가 (Self-Assessment):
1. 이번 반기 가장 자랑스러운 성과는?
2. 목표 대비 달성률은? (OKR 기준)
3. 개선이 필요한 영역은?
4. 다음 반기 집중할 목표는?

상향 피드백 (Upward Feedback):
1. 팀장이 더 해줬으면 하는 것은?
2. 팀장이 그만했으면 하는 것은?
3. 팀장이 계속했으면 하는 것은?

성과 등급 (Performance Rating):
- Exceeds Expectations (상위 10%)
- Meets Expectations (60-70%)
- Developing (20-25%)
- Below Expectations (5% 이하, PIP 대상)
```

## 보상 체계 (Compensation)

### 급여 밴드 설계

```
레벨 구조 예시 (스타트업):
L1: Junior         — ₩30-45M
L2: Mid-level      — ₩45-65M
L3: Senior         — ₩65-90M
L4: Staff/Lead     — ₩90-120M
L5: Principal      — ₩120-160M
L6: Director+      — ₩160M+

밴드 설계 원칙:
- 시장 P50 기준 중간값 설정 (Radford, 원티드 등 벤치마크)
- 밴드 내 범위: 중간값 ±20%
- 밴드 간 겹침 허용 (경력자 승진 시 급여 역전 방지)
```

### 총 보상 패키지 구성

| 항목 | 스타트업 | 중견기업 |
|---|---|---|
| 기본급 | 시장 P50 | 시장 P50-75 |
| 성과급 | 연 0-30% | 연 10-20% |
| 스톡옵션 | 0.01-1% (직급별) | 제한적 |
| 4대보험 | 필수 | 필수 |
| 식대/교통비 | ₩200K/월 | ₩200K/월 |
| 자기계발비 | ₩1-3M/년 | ₩500K-2M/년 |
| 재택/유연근무 | 높음 | 중간 |

## 조직 문화 & 인게이지먼트

### eNPS (Employee Net Promoter Score)

**측정 방법:** "이 회사를 친구/지인에게 일하기 좋은 곳으로 추천하겠습니까?" (0-10점)
- Promoters: 9-10점
- Passives: 7-8점
- Detractors: 0-6점

**eNPS = Promoters% - Detractors%**

| Score | 해석 |
|---|---|
| 50+ | 탁월 |
| 20-49 | 양호 |
| 0-19 | 개선 필요 |
| 0 미만 | 위험 |

### 온보딩 체크리스트 (첫 90일)

**Day 1:**
- [ ] 업무 장비 및 계정 세팅 완료
- [ ] 팀 소개 미팅 (1:1 + 팀 전체)
- [ ] 회사 미션/비전/전략 오리엔테이션

**Week 1:**
- [ ] 주요 협업 도구 교육 (Slack, Notion, JIRA 등)
- [ ] 첫 업무 태스크 배정
- [ ] 팀장과 기대치 정렬 미팅

**Month 1:**
- [ ] 30일 체크인 (팀장 + HR)
- [ ] 첫 성과물 완료
- [ ] 주요 이해관계자 파악 완료

**Month 3:**
- [ ] 90일 리뷰 (성과 + 문화 적합도)
- [ ] 다음 분기 OKR 설정
- [ ] 온보딩 경험 피드백 수집
